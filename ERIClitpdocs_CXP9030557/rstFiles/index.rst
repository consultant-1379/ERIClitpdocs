.. _litp-overview:

--------
Overview
--------
The Linux IT Platform (**LITP**) is designed to be a lightweight, open, plugin
based platform.
The sections below provide more information for plugin developers, REST
developers and those involved in installing, deploying and configuring **LITP**

---------------------------------
Information for Plugin Developers
---------------------------------
The :ref:`litp-sdk` page is the starting point for those looking to create plugins for **LITP**.It provides the following useful resources

* :ref:`plugin-dev-doc` for plugin developers, including a Developer's Guide and API description
* :ref:`deployment-model-info` for information on the core Deployment Model structure
* :ref:`reference-doc` for additional reference material
