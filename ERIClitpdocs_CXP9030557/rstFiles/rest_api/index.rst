.. _rest-api:

=========================
 LITP |release| REST API
=========================

-------
 Scope
-------
This document is the REST Application Programming Interface (API) Specification
for LITP 2.0. It describes the details of the REST API, request and response
formats and error handling.

------------------
 Document Purpose
------------------
This document describes the specifications of the REST API and focuses on the
format of HTTP requests and responses sent to or from the LITP REST API for
CREATE, READ, UPDATE and DELETE (CRUD) operations and item discovery operations.
