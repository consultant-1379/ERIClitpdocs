Setup the Plugin Development Environment
===========================================

The document provides the information you need to set up a plugin development 
environment. It describes the hardware and software dependencies, and provides information 
on how to clone LITP Maven Archetypes and build plugin and model extension templates.

An archetype is a Maven project templating toolkit used to create a template model extension or a 
plugin project.
